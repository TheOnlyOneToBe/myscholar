## 📝 Commandes Artisan Principales

### Configuration
```bash
# Initialiser un nouveau client
php artisan client:initialize

# Installer tous les modules
php artisan client:initialize --all

# Installer des modules spécifiques
php artisan client:initialize --modules=Students,Classes,Grades
```

### Base de Données
```bash
# Voir l'état des migrations
php artisan migrate:status

# Exécuter les migrations
php artisan migrate

# Revenir en arrière
php artisan migrate:rollback
```

### Utilitaires
```bash
# Accéder à la console PHP
php artisan tinker

# Lancer le serveur de développement
php artisan serve

# Voir tous les rôles
php artisan roles:list

# Voir toutes les permissions
php artisan permissions:list
```

Pour plus de détails, voir [GUIDE_INSTALLATION.md](./GUIDE_INSTALLATION.md#commandes-artisan-principales)

## 🔐 Verification Checklist for Dashboard

Before allowing access to each feature, verify:

```php
// 1. Student role verified
if (!$user->hasRole('student')) {
    return error_response;
}

// 2. Student record exists
$student = Student::where('user_id', $user->id)->first();
if (!$student) {
    return error_response;
}

// 3. Module is active and installed
if (!$moduleManager->canUseModule('Grades')) {
    return error_response;
}

// 4. Required tables exist
if (!Schema::hasTable('grades')) {
    return error_response;
}

// 5. Data belongs to student
if ($grade->student_id !== $student->id) {
    return error_response;
}
```
